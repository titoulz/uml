# 🔊 Instructions pour le son "Salaam Aleykum"

## 📁 Placement du fichier audio

Placez votre fichier audio dans ce dossier (`/fx/`) avec un des noms suivants :
- `salaamaleykum.mp3` (recommandé)
- `salaamaleykum.wav`
- `salaamaleykum.ogg`

## ✅ Le système testera automatiquement ces formats dans l'ordre

## 🎯 Déclenchement du son

Le son "Salaam Aleykum" se jouera automatiquement dans ces situations :

1. ✅ **Connexion réussie** - Quand un utilisateur se connecte avec succès
2. ✅ **Inscription réussie** - Quand un nouvel utilisateur s'inscrit

## ⚙️ Configuration

- **Volume** : 70% (modifiable dans `auth-script.js`)
- **Délai** : 1.5 secondes avant redirection pour laisser le temps au son de se jouer
- **Formats supportés** : MP3, WAV, OGG

## 🧪 Test

Pour tester :
1. Placez votre fichier audio dans ce dossier
2. Ouvrez `auth.html` dans votre navigateur
3. Connectez-vous avec : `test@exemple.com` / `test123`
4. Le son devrait se jouer ! 🎵

## 🌟 Ambiance Maghrébine

Ce son "السلام عليكم" (Salaam Aleykum) ajoute une touche authentique et chaleureuse à l'expérience utilisateur, renforçant l'atmosphère maghrébine du site !
